# Spots

Spots is a social media website used for posting pictures for the world to see. It was created using HTML and CSS based on a design brief from figma.

## Features/Technology

- Semantic HTML
- CSS
- Flexbox
- Grid
- Positioning
- Responsive Design
- Hover effects
